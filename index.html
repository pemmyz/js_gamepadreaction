Press an Arrow Key, WASD, or a Gamepad Face Button to Join!

Pause/Resume: SPACE or Gamepad 'Start' Button

Reset Game: 'R' Key

Gamepad: Not Detected

Help & Game Modes

Select a game mode. Changing modes will reset the current game.

Competitive Mode (Default)

Be the first player to hit the correct button to score a point. The game speeds up on correct hits and slows down on misses. It's a race!

Co-op Mode

All active players must press the correct button before the timer runs out. If successful, a new prompt appears. If the timer runs out, anyone who didn't press gets a "Miss".

Dropout: Miss 3 prompts in a row (by timeout) and you will be dropped out. You can rejoin at any time by pressing one of your buttons.
